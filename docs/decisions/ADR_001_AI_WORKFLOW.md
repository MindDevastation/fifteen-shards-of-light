# ADR_001_AI_WORKFLOW

## Status
Accepted

## Context
This project is a small personal gift game with a fixed scope and a strong need to avoid uncontrolled feature growth.

## Decision
Use an AI-assisted workflow built around small, explicit task slices. Each task must include repository inspection, a short plan, a test-impact check, scoped changes, and a handoff report.

## Consequences
- Development remains focused on the 15–20 minute gift build.
- Documentation and prompts help preserve canon and constraints between tasks.
- Large systems and unrequested features are intentionally discouraged.

# Fifteen Shards of Light

A small personal non-commercial 3D puzzle-platformer gift game built with Godot 4.x and GDScript.

## Current Stage
Stage 0B — AI Workflow Scaffold.

The Godot project was created manually through Godot and should not be recreated from scratch.

## Project Canon
- Nine-tailed fox.
- Soul sphere.
- 15 soul shards.
- 15 short micro-levels.
- Couplets after each level.
- Hidden phrase: “Алена, я люблю тебя”.
- Gentle final epilogue.

## Production Goal
Create a finished playable 15–20 minute gift version. This is a one-time gift build, not a commercial product.

## Production Pipeline
1. Stage 0 — GitHub, AI workflow, local environment.
2. Stage 1 — playable greybox skeleton.
3. Stage 2 — model/visual polish.
4. Stage 3 — sound pass.
5. Stage 4 — final gift polish.
6. Stage 5 — gift build.

## Scaffold Map
- `docs/req` — requirements and project canon.
- `docs/spec` — stage and feature specifications.
- `docs/architecture` — system maps and architecture notes.
- `docs/qa` — handoff and manual test checklists.
- `docs/decisions` — ADR records.
- `prompts/codex` — Codex task prompts.
- `prompts/review` — QA review prompts.
- `prompts/reusable` — reusable prompt fragments.
- `scenes/core`, `scenes/levels`, `scenes/ui` — future Godot scenes.
- `scripts/core`, `scripts/player`, `scripts/soul`, `scripts/ui` — future GDScript files.
- `assets/models`, `assets/materials`, `assets/audio`, `assets/fonts`, `assets/placeholders` — future assets.
- `manual_tests` — human-run checks.
- `handoff` — task handoff notes.

## Scope Guardrails
This project intentionally avoids combat, inventory, dialogue systems, online features, open world structure, complex AI, and complex RPG systems. Polished assets should wait until the greybox build works.
